# 本章概览

本章介绍 Rust 最核心的特性——**所有权系统**。你将学到：

- **内存与数据流动**：理解栈与堆的区别，掌握移动、拷贝与克隆三种数据交互方式
- **所有权规则**：三条黄金规则、变量作用域、String 类型与函数交互
- **什么是所有权**：所有权规则的完整解释和应用
- **概念辨析**：所有权与可变性、变量遮蔽、两种字符串类型的区别
- **引用与借用**：在不转移所有权的情况下使用数据
- **切片（Slice）**：对序列数据的局部引用

学完本章，你将理解 Rust 如何在没有 GC 的情况下保证内存安全，以及如何在实际代码中正确管理数据的所有权。

接下来，我们先从**内存与数据流动**开始，理解栈和堆的区别，以及移动、拷贝与克隆这三个基础概念。
