## 本章概览

学一门语言，第一步永远是同一件事：弄清楚它的基本构成要素——变量怎么声明，类型系统什么样，怎么写条件分支和循环，怎么定义函数。

这些东西在各门语言里大同小异，但 Rust 有几处独特的地方值得注意。比如变量默认不可变，比如 `if` 是表达式而不是语句，比如函数的返回值是最后一个表达式的值而不是 `return`。这些设计选择背后都有原因，理解了原因，规则就不难记住。

本章把这些基础语法挨个讲清楚，外加属性和宏两个 Rust 特有的概念。

## 一段代码，看懂本章要学什么

下面这段代码虽然不长，但用到了本章涵盖的大部分内容——变量、数据类型、函数、控制流、循环和格式化输出：

```rust runnable
// 根据分数判断等级（函数 + 控制流）
fn grade(score: u32) -> &'static str {
    if score >= 90 {
        "优秀"
    } else if score >= 80 {
        "良好"
    } else if score >= 60 {
        "及格"
    } else {
        "不及格"
    }
}

fn main() {
    // 变量与数组
    let scores = [95, 82, 61, 45];

    // for 循环遍历
    for score in scores {
        // 格式化输出（println! 宏）
        println!("分数 {:>3}：{}", score, grade(score));
    }
}
```

读完本章，你会知道这段代码里每一行是什么意思，也能自己写出类似的程序。

## 本章涵盖的内容

**01 — 注释**
单行注释 `//`、多行注释 `/* */` 和文档注释 `///` 的写法与用途。

**02 — 格式化输出**
`println!`、`print!`、`format!` 的各种占位符格式，包括对齐、精度控制和调试输出 `{:?}`。

**03 — 数据类型**
整数、浮点数、布尔值、字符四种标量类型，以及元组和数组两种复合类型。Rust 是静态强类型语言，类型在编译时确定。

**04 — 变量**
`let` 绑定与默认不可变性，`mut` 关键字，变量遮蔽（shadowing），以及 `const` 常量的用法。

**05 — 控制流**
`if` 表达式（注意：是表达式，能返回值）、`loop` / `while` / `for` 三种循环，循环标签与 `break` 返回值。

**06 — 函数**
函数定义、参数类型标注、返回值语法（`->` 符号），以及 Rust 中语句与表达式的区别——这是新手最容易踩坑的地方。

**07 — 属性**
`#[attribute]` 和 `#![attribute]` 两种语法形式，`#[allow(dead_code)]` 禁用警告，`#[cfg(...)]` 条件编译。

**08 — 宏**
宏与函数的区别，用 `macro_rules!` 定义声明宏，以及自定义 derive 宏、类属性宏、类函数宏三种过程宏的基本概念。

**09 — 综合练习**
5 道经典算法题，综合运用本章所学，包括斐波那契数列、质数判断、汉诺塔等。
