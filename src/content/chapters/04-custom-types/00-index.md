你已经学过 Rust 的基本类型（整数、浮点、布尔、字符串等）。现在是时候创建自己的类型了。

Rust 提供了强大的类型系统，让你能够以安全、表达性强的方式组织和处理数据。本章将介绍：

1. **结构体** — 将相关的数据组织在一起
2. **方法与关联函数** — 给数据类型附加行为
3. **枚举** — 表示一个值是某个离散集合中的一种
4. **Option\<T\>** — 安全地处理值的存在或缺失
5. **模式匹配** — 强大的控制流工具
6. **常量与静态变量** — 编译期已知的值
