你已经学过 Rust 的基本类型（整数、浮点、布尔、字符串等）。现在是时候创建自己的类型了。

结构体用于将相关数据组织在一起，枚举表达"多选一"的语义，`match` 则是处理枚举值的利器——这三者结合，构成了 Rust 描述复杂问题域的核心工具。`Option<T>` 是 Rust 处理"值可能不存在"问题的答案，用来彻底替代其他语言中的 `null`。
