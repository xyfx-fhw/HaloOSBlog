# 章节概览

你已经掌握了 Rust 的基础类型和自定义类型。现在是时候学习 Rust 标准库中最常用的**集合类型**了。

在实际开发中，你很少使用单个值。大多数时候，你需要管理一组数据。本章介绍了 Rust 标准库中三种最重要的集合类型，它们几乎出现在每个 Rust 程序中：

1. **Vec\<T\>** — 可动态增长的数组，用于存储一系列相同类型的值
2. **String** — 可变字符串，掌握 String 与 &str 的区别（我们再深入一些细节）
3. **HashMap\<K, V\>** — 键值对集合，通过键快速查找值
