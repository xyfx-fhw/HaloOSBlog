Rust 的类型系统远不止于"声明变量的类型"。本章深入四个主题：编译器如何在大多数场景下自动推断类型，`as` 关键字如何进行显式数值转换，类型别名如何增强代码可读性，以及 newtype 模式如何用零开销的方式创建出语义不同的新类型——让编译器帮你区分「米」和「厘米」。
