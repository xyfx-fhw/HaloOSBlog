# 模块系统

Rust 提供了强大的**模块系统**来帮助你组织和管理代码。本章包含四个核心主题：

1. **Package 和 Crate** — 理解编译单元和项目组织的基本概念
2. **模块与可见性** — 学会用模块组织代码，用 `pub` 控制可见性
3. **路径与 use** — 掌握在模块树中导航和访问项
4. **综合练习** — 设计完整的库结构

通过这一章，你将学会设计清晰、可维护的 Rust 项目结构。
