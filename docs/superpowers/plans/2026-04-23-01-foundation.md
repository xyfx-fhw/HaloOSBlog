# 子计划 1：框架基础（Foundation）实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标：** 搭建可运行的 Astro 5 项目框架，包含全局深色主题样式、顶部导航栏、首页、章节列表页和基础章节文档页（无侧边栏），访问 `http://localhost:4321` 可看到完整的静态网站。

````markdown
## 为什么选择 Rust？

Rust 是一门系统编程语言，核心目标是：**在没有垃圾回收器的前提下，保证内存安全**。

传统系统语言（C/C++）给程序员极大控制权，但容易引入内存错误（空指针、悬垂指针、缓冲区溢出）。Rust 通过**所有权系统**在编译期检查这些错误，零运行时开销。

## Rust 适合做什么？

Rust 在以下领域表现出色：

- **系统软件**：操作系统内核、设备驱动、嵌入式开发
- **WebAssembly**：在浏览器运行接近原生性能的代码
- **命令行工具**：`ripgrep`、`fd`、`bat` 等知名工具均由 Rust 编写
- **网络服务**：高性能 Web 服务器与异步运行时（Tokio、Actix-web）

## 核心特性一览

| 特性 | 说明 |
|------|------|
| 所有权系统 | 编译期内存管理，无 GC，无悬垂指针 |
| 借用检查器 | 防止数据竞争，保证并发安全 |
| 零开销抽象 | 高级语言特性不带运行时代价 |
| 强类型系统 | 代数数据类型、trait、泛型 |
| 优秀工具链 | Cargo 包管理器、rustfmt、Clippy |
````

```astro
import BaseLayout from '../../components/layout/BaseLayout.astro';
import { getCollection } from 'astro:content';

const allChapters = await getCollection('chapters');
allChapters.sort((a, b) => a.slug.localeCompare(b.slug));

// 按目录前缀（第一段路径）分组
const groupMap = new Map<string, typeof allChapters>();
for (const entry of allChapters) {
  const groupKey = entry.slug.split('/')[0];
  if (!groupMap.has(groupKey)) groupMap.set(groupKey, []);
  groupMap.get(groupKey)!.push(entry);
}

const groups = Array.from(groupMap.entries()).map(([key, entries]) => ({
  key,
  index: entries.find(e => e.slug.endsWith('/00-index')),
  articles: entries.filter(e => !e.slug.endsWith('/00-index')),
```
